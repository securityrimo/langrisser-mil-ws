# langrisser-mil-ws
================================================================
 랑그릿사 밀레니엄 WS - The Last Century  한글패치 v0.9
 (Langrisser Millennium WS - Korean Translation Patch)
================================================================

■ 소개
  원더스완 컬러용 「랑그릿사 밀레니엄 WS」의 일본어 → 한국어 번역 패치입니다.

■ 적용 방법 
  1) IPS 패치 적용
     - 원본 ROM(1MB, "랑그릿사 밀레니엄 WS - The Last Century.ws")에
       「랑그릿사_밀레니엄_한글.ips」를 적용
     - 패치 도구: Lunar IPS(Flips), NINJA 등

  2) xdelta 패치 적용
     - 원본 ROM에 「랑그릿사_밀레니엄_한글.xdelta」를 적용
     - 패치 도구: xdelta UI, DeltaPatcher 등

■ 해시값 (무결성 확인)
  ▶ 원본 ROM (패치 적용 대상 / 1,048,576 bytes)
     - CRC32 : 9E0854E2
     - MD5   : 0E4369D3F611377433272B3563325CA2
     - SHA-1 : 9FDF3F5821F2A4488139525FE49E87B23DE4571F
  ▶ 패치 후 ROM (정상 적용 결과 / 1,048,576 bytes)
     - CRC32 : 8A7DF807
     - MD5   : 18279B1C693CD6EA1EE229CF21472E9E
     - SHA-1 : F1B35E78D4E7F85D5E52E8008F874BEABFE65A56
  ※ 원본 CRC32가 위와 다르면 판본/덤프가 다른 것이니 패치가 맞지 않을 수 있습니다.
     패치 적용 후 결과물의 CRC32가 위 값과 일치하면 정상 적용된 것입니다.
  ▶ 패치 파일 자체
     - IPS    : CRC32 BB4F227B / MD5 9CD472252267E91BD96C6CE71BBCE6DA
     - xdelta : CRC32 4180570C / MD5 82D67DD886D412E14375FD0A60805290

■ 알려진 제한 (v0.9)
  - 소형 상태창의 캐릭터 이름/클래스는 그래픽 타일이라 아직 일본어로 표시됩니다.
  - 일부 화면은 전각(넓은) 띄어쓰기로 표시됩니다.
  - 캐릭터명 음차는 최대한 통일했으나 일부 편차가 남아있을 수 있습니다.

■ 기술 정보
  - 체크섬 자동 보정, ROM 크기 불변(1MB)

■ 라이선스/주의
  - 개인 학습·연구 목적. 원본 ROM은 정당하게 소유한 것만 사용하세요.
  - 재배포 시 본 안내문을 포함해 주세요.
  - 원칙적으로, 해당기종에 원본 롬 카트리지를 보유한 분이 개인적으로만 이용한다는 전제 하에 본 한글패치 파일을 다운로드할 수 있습니다.
  - 상기 조건(A)에 해당하지 않는 분이 본 한글패치 파일을 다운로드하여 이용할 경우에 발생하는 모든 책임은 이용 당사자에게 있으며, 패치 제작자는 어떠한 책임도 지지 않습니다.
  - 본 한글패치 파일 자체를 제작자에 허가없이 타 사이트에 재배포하지 말아 주시고, 재배포를 원하실 경우 패치원본 게시물의 링크를 공유해 주시기 바랍니다. (아카이빙 포함)
  - 단, 제작자의 사망이 확인된 경우에는 예외로 합니다. (제 3자 통보후 3년 혹은 마지막 활동기록후 5년) ﻿
  - 모든 저작권은 게임 개발사와 유통사에게 있으며 패치 제작자는 개발사와 유통사에게 어떤 권리도 주장하지 않습니다.
